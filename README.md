### Getting Started and Installing

1. Clone the repo
```sh
https://github.com/iobasa/Al-Osman_Azores_Kaur_Obasa_FIP.git
```
2. Install NPM packages
```sh
npm install
```
3. Import database to MAMP or WAMP from the 'database' folder
4. Run Node
```sh
npm start
```
